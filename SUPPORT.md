# Support

GitHub Issues are the support channel for self-hosted ReproIt Cloud. Include the
Cloud version, deployment method, database version, storage backend, redacted
logs, and reproduction steps. Security reports follow [SECURITY.md](SECURITY.md).

The supported 1.x deployment is the documented Compose stack or an equivalent
Linux container deployment with PostgreSQL and durable local or S3-compatible
evidence storage. Operators own capacity, monitoring, backups, TLS, and upgrades.
